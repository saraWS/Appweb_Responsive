TYPESCRIPT

Es un lenguaje de programación desarrollado por Microsoft. Es un superconjunto tipado de JavaScript.

ANGULAR

Es un framework de desarrollo web de código abierto creado por Google, se utiliza principalmente para crear aplicaciones web de una sola página (aplicaciones que cargan una única página HTML y actualizan dinámicamente el contenido a medida que el usuario interactúa con la app).

IONIC

Es un framework de desarrollo de aplicaciones móviles y web, permite a los desarrolladores crear aplicaciones híbridas.
